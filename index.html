git init: Inicializa un nuevo repositorio Git.
git clone: Clona un repositorio existente en un nuevo directorio.
git add: Agrega cambios al área de preparación.
git commit: Guarda los cambios en el repositorio.
git status: Muestra el estado de los archivos en el directorio de trabajo y el área de preparación.
git push: Sube los cambios locales al repositorio remoto.
git pull: Obtiene cambios desde el repositorio remoto y los fusiona en el repositorio local.
git branch: Lista, crea o borra ramas.
git merge: Fusiona cambios de una rama a otra.
git log: Muestra un historial de confirmaciones.
git diff: Muestra las diferencias entre commits, archivos o ramas.
git remote: Muestra los repositorios remotos configurados.
git fetch: Descarga cambios desde el repositorio remoto sin fusionarlos.
git checkout: Cambia entre ramas o versiones de archivos.
git reset: Deshace cambios y mueve la cabeza de la rama a un commit específico.

Etiquetas Semánticas:


article: Define un artículo independiente.
section: Define una sección en un documento.
header: Define la cabecera de un documento o sección.
footer: Define el pie de un documento o sección.
nav: Define una sección de navegación.

Etiquetas de Texto:


p: Define un párrafo.
h1 a h6: Define encabezados de diferentes niveles.
strong: Hace que el texto sea fuerte o importante.
em: Enfatiza el texto.

Etiquetas de Agrupación:


div: Define una división o sección genérica.
span: Define una pequeña sección en línea.


Etiquetas Multimedia:


img: Inserta una imagen.
audio: Incorpora contenido de audio.
video: Incorpora contenido de video.

Etiquetas de Tablas:


table: Define una tabla.
tr: Define una fila en una tabla.
td: Define una celda de datos en una tabla.
th: Define una celda de encabezado en una tabla.

Etiquetas de Formularios:


form: Define un formulario.
input: Define un campo de entrada en un formulario.
select: Define una lista desplegable.
textarea: Define un área de texto de varias líneas.
